Dispositivo: Roteador

Conceitos rápidos

  • Roteador: Encaminha dados entre redes, usando tabelas de roteamento.
  • Switch: Gerencia tráfego local com MAC addresses, evitando colisões.
  • Hub: Replica dados para todos os ports, causando broadcasts.
  • Modem: Converte sinais digitais/analógicos, essencial para DSL ou cabo.
  • Firewall: Filtra pacotes com regras baseadas em IP/portas.
  • Repetidor: Amplifica sinal na camada física para estender alcance.
  • Access Point: Expande Wi-Fi, conectando sem fio a rede cabeada.
  • Bridge: Conecta segmentos de rede, filtrando tráfego por MAC.
