Dispositivo: Roteador

Conceitos rápidos

  • Roteador: Encaminha dados entre redes.
  • Switch: Gerencia tráfego local eficiente.
  • Hub: Replica dados para todos (menos eficiente).
  • Modem: Converte sinais para provedor.
  • Firewall: Protege contra ameaças.
  • Repetidor: Amplifica sinal para maior alcance.
